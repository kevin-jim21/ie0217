Detailed Description

Esta estructura representa el juego "Ahorcado".

En esta estructura se almacenan todos los datos necesarios para definir el "status" del juego durante su desarrollo y determinar cuando finaliza. Se van a utilizar punteros hacia la estructura para actualizar sus valores durante el juego.
