Detailed Description

Esta clase se utiliza para evaluar si una dirección correo electrónico es válido o no.

Esta clase será instanciada en el main, para por medio de su método público validarCorreo(), recibir las posibles direcciones dadas por el usuario en formato string y validarlas.
