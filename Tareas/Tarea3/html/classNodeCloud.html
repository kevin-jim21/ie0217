Detailed Description

Clase para instanciar nodos de la lista entrelazada.

Esta clase se encarga de crear todos aquellos nodos los cuales tendran copias de los datos de la HashTable y se almacenaran en una nube. Todos ellos seran entrelazados de forma lineal y tendran un puntero para acceder al nombre y número telefónico que le corresponde en la HashTable.
