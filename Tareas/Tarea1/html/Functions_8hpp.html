◆ CheckGame()

int CheckGame ( Ahorcado * Game)

Revisar cada turno si el juego ya terminó.

Esta función se encarga de chequear primero si la palabra ya fue adivinada, en caso de que no sea así, consultará si el usuario alcanzó el límite de intentos utilizados; en caso de que sea así le dará al usuario una última oportunidad de adivinar la palabra. En cualquiera de estos casos, la función retorna 1 para indicar que el juego ya finalizó, mientras que si no se cumple ninguno de estos casos se retorna 0 para indicar que el juego debe continuar. La función siempre se llama después de salir de la función GuessWord y determina cuando finaliza el bucle.

Parameters
GameEstructura principal del juego.
Returns
Número que indica si el juego terminó o no.

void GuessWord ( Ahorcado * Game,
char character )

Función para ingresar letras y verificar si el usuario adivinó.

Esta función extrae la palabra a adivinar de la estructura, la convierte en arreglo y lo recorre con un bucle comparando con el carácter que ingresó el usuario. Después de verificar que el carácter aún no había sido ingresado en el arreglo de la palabra adivinada, se coloca la letra en el arreglo de letras adivinadas. Si la letra no es adivinada, suma uno a los intentos realizados por el usuario. Esta función va a ser llamada dentro de un bucle después haber inicializado el juego con la función StartGame.

Parameters
GameEstructura principal del juego.
characterCualquier letra ingresada por el usuario.

void StartGame ( Ahorcado * Game,
std::vector< std::string > dict )

Función que inicializa el juego.

Esta función elige una palabra aleatoriamente de un diccionario, incializa el estado de la palabra adivinada con caracteres "_" y por último, inicializa en 0 el número de intentos realizados. Esta función va a ser llamada exclusivamente al ingresar desde el menú a la opción de jugar.

GameEstructura principal del juego.
