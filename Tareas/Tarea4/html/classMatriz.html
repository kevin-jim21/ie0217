Detailed Description

template<typename Tipo>
class Matriz< Tipo >

Template class mediante la cual se definen matrices y se realizan operaciones para los tipos de datos int, float y std::complex.

En esta template class se encuentran los métodos necesarios para rellenar la matriz y realizar operaciones entre ellas. Los métodos para dimensionar y rellenar la matriz utilizan control de excepciones propios de la STL para evitar la caída del programa ante el ingreso de un dato erróneo por parte del usuario.
